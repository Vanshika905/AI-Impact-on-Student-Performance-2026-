# AI-Impact-on-Student-Performance-2026
An end-to-end Machine Learning project analyzing how AI tools like ChatGPT, Claude, and GitHub Copilot influence student productivity, academic performance, and career confidence.

## 📌 Project Overview
This project investigates the real-world impact of AI adoption among students using a synthetic educational dataset.
The analysis explores:
- AI usage behavior
- GPA changes after AI adoption
- Time saved using AI tools
- Ethical concerns surrounding AI use
- Career readiness and confidence
The project combines:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning
- Clustering
- Explainable AI (SHAP)


## 🎯 Objectives
- Understand whether AI tools improve academic outcomes
- Analyze productivity gains from AI usage
- Predict career confidence using ML models
- Identify different student behavior patterns using clustering
- Improve model transparency using SHAP explainability

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
- SHAP

# 📊 Key Insights
- AI improves productivity more consistently than GPA
- Time saved is a stronger predictor of career confidence
- Heavy AI usage does not always lead to better outcomes
- Effective AI usage matters more than frequency
- SHAP explainability helps identify trustworthy model behavior
